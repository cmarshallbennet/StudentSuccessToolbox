Spacers

Spacers are used to add margins to elements. It used ems so is based on parent size

Spacer Top classes

.spacer-top (1em)
.spacer-top-2x (2em)
.spacer-top-3x (3em)
.spacer-top-4x (4em)

Available classes

Use a combination of spacer classes to add margin. Spacers spacer-xs-top-* and spacer-top-* are the same. Use spacer-*-top-0 to remove top margin for different screen sizes. For example the class <div class="spacer-xs-top-2x spacer-md-top-0"></div> will have a 2em top margin for small devices and tablets and no margin for Desktops;

Extra small devices Phones (<768px) Small devices Tablets (≥768px) Medium devices Desktops (≥992px) Large devices Desktops (≥1200px)
.spacer-xs-top-* Spacer Spacer Spacer Spacer
.spacer-sm-top-* none Spacer Spacer Spacer
.spacer-md-top-* None None Spacer Spacer
.spacer-lg-top-* None None None Spacer
* values 0, 2x, 3x or 4x

Spacer Bottom classes

.spacer-bottom (1em)
.spacer-bottom-2x (2em)
.spacer-bottom-3x (3em)
.spacer-bottom-4x (4em)

Slider Control

The slider use is a modified version of rzslider with better accessibility. There are 2 types of slider design. Default which is circle and square. All you need to do is add a class .square to the slide to make it square.

You can easily change the colors of by setting a border color eg border-color: #f9a029; of .rz-pointer.

<style> .slider-orange .rz-pointer{ border-color: #f9a029; } </style>
<style> .slider-blue .rz-pointer{ border-color: #08a7cd; } </style>
